Add accessible per-product FAQs and custom product tabs to WooCommerce.

== Description ==

WooCommerce Product FAQ adds repeatable FAQs and custom tabs to the product editor. FAQs can appear in a product tab or after the product summary.

== Installation ==

1. Upload the plugin directory to `/wp-content/plugins/`.
2. Activate WooCommerce.
3. Activate WooCommerce Product FAQ.
4. Edit a product and open the FAQ panel.
